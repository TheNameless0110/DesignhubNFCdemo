# Design Hub — NFC Virtual Business Card

> A premium, interactive NFC virtual business card built with **React + Vite + Tailwind CSS** featuring a stunning **Claymorphism** UI.

![Design Hub NFC Card](https://images.unsplash.com/photo-1556742049-0cfed4f6a45d?auto=format&fit=crop&w=800&q=80)

## ✨ Features

- **3D Claymorphism UI** — Rich layered shadows and organic blob shapes
- **Animated Background** — GPU-accelerated floating blobs with breathe animation
- **Expandable Services Accordion** — Smooth CSS grid-row animation
- **Contact Links** — Email, Phone, Website with tap-to-action
- **Social Links** — Instagram, LinkedIn, Twitter, Dribbble
- **NFC Ready** — Designed to be served via an NFC tap URL
- **Mobile Optimized** — Responsive, no tap-highlight flash

## 🚀 Getting Started

```bash
# Install dependencies
npm install

# Start dev server
npm run dev

# Build for production
npm run build
```

## 📇 NFC Setup

1. Build the project: `npm run build`
2. Deploy the `dist/` folder to any static host (Vercel, Netlify, GitHub Pages)
3. Encode your deployed URL onto an **NTAG215** NFC card
4. Tap the card on any modern smartphone — no app required!
